# オープンソース英単語帳『ブラックジャックによろしく』

本リポジトリは、マンガ『ブラックジャックによろしく』を題材としたオープンソースの英単語帳を作成するために用います。

## プロジェクトページ

プロジェクトの詳細については[プロジェクトページ](https://error96num.notion.site/da97d1b47d544ac1bb27b8aa20748db1)をご参照ください。

下記、作成協力も承っております！

* セリフ原稿の書き起こし
* 単語帳への登録を希望する英語表現の提出
* 単語帳に添付する英文音声の録音
* 誤りの訂正
